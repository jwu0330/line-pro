# Privacy Policy for Open LINE in Edge (Pro)

Last updated: 2025-11-24

Open LINE in Edge (Pro) is a Chrome extension designed solely to allow users to open
Microsoft Edge and automatically activate the LINE extension within Edge.  
This extension does **not** collect, store, transmit, sell, or share any personal data.

## 1. Data Collection
This extension does **not** collect any type of user data, including but not limited to:
- Personal identifying information
- Communications data
- Browsing history
- Website content
- Location data
- Financial or authentication information
- Usage behavior or activity tracking

## 2. How the Extension Works
The extension uses Chromeâ€™s official Native Messaging API to execute a local PowerShell script.
This script performs only the following actions:
1. Launch Microsoft Edge
2. Perform automated UI interaction to click the LINE extension icon

The Native Host does *not* read user data, monitor browsing activity, log information,
or communicate with external servers.

## 3. Third-Party Sharing
Since no data is collected, no information is shared or transferred to any third party.

## 4. Compliance
The extension complies with all applicable Chrome Web Store policies, including:
- Single Purpose Policy
- Limited Use Requirements
- User Data Privacy and Security Policies
